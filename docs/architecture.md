# Nexus Architecture

Nexus follows a modular architecture.

Core
Handles assistant logic and voice interaction.

Automation
Controls system actions such as launching apps and system power operations.

File System
Handles file creation, deletion, searching, and folder organization.

Services
Fetches external information like weather, sports scores, and news.

UI
Provides graphical interface and assistant animations.

Utils
Contains reusable helper utilities used throughout the system.
